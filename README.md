
# 武汉家教讯息中心——猿辅导
## 一.项目名称：猿辅导武汉家教网
* ![猿辅导项目图案](https://raw.githubusercontent.com/BTB8059/wust1-teaching/master/src/main/webapp/img/yuanfudao.jpg)
## 二.项目说明：
* 为家长学生提供可靠高质量范围广的家教信息
* 为寻求兼职家教的老师学生提供信息发布的平台
* 设置反馈，提供家教评价，为其他需求者提供参考
